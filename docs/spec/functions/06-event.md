## 7.1 定義

トランザクションを横断的に束ねるグルーピング概念。

## 7.2 属性

| 属性名     | 型     | 必須 | 説明       |
| ---------- | ------ | ---- | ---------- |
| id         | ID     | ○    | 一意識別子 |
| name       | 文字列 | ○    | イベント名 |
| occurredOn | 日付   | ○    | 発生日     |

## 7.3 仕様

- 年度を跨いでトランザクションを束ねられる
- 既存トランザクションへの後付け紐付けが可能
- 名称変更が可能
- 紐づくトランザクションが1件以上存在するイベントは削除不可

## 7.4 用途例

- 旅行（金沢旅行、海外旅行）
- 家電購入
- 給与一式
- 特定プロジェクトの支出

## 7.5 削除ルール

- イベント削除は物理削除とする
- 紐づくトランザクションが存在しない場合のみ削除できる
