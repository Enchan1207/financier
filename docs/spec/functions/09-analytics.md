## 10.1 ダッシュボード

| 表示項目       | 説明                     |
| -------------- | ------------------------ |
| 内部残高       | 全期間の収支差分         |
| 積立合計       | 全積立定義の残高合計     |
| 年度収支サマリ | 当年度の収入・支出・差分 |

## 10.2 円グラフ

- 年度単位でカテゴリ構成比を表示
  - 収入カテゴリ構成比
  - 支出カテゴリ構成比
- 円グラフは構成比のみを表す

## 10.3 棒グラフ

- カテゴリ別に実績・予算・差分を表示
- 年度単位・月次フィルタの両方に対応

## 10.4 一覧・サマリ

- 年度収支サマリ
- 積立進捗一覧
- カテゴリ別金額・割合・予算差分
- 年度別内部残高推移

## 10.5 イベント別集計

- イベント単位の合計金額
- カテゴリ内訳
- 年度別内訳

## 10.6 年度間比較

- 複数年度の収入・支出・差分の推移をグラフ表示
- カテゴリ別の年度間比較（同一カテゴリの各年度実績）
- 年度ごとの伸び率・減少率の算出

## 10.7 財政体力の推移

- 年度別の内部残高推移をグラフ表示
- イベント単位の影響分析（特定イベントが内部残高に与えた影響を可視化）
- 月次での内部残高推移（オプション）

## 10.8 投資判断支援

- 現在の内部残高表示
- 全積立定義の充足率一覧表示
- 仮想支出額を入力した場合の内部残高シミュレーション
- 積立充足状況を考慮した余裕度の表示
