## 10.1 ダッシュボード

| 表示項目         | 説明                                                       |
| ---------------- | ---------------------------------------------------------- |
| 実質自由資金     | 内部残高 − 積立残高合計。主指標として大きく表示する        |
| 積立残高合計     | 全積立定義の残高合計。実質自由資金のサブ情報として並記する |
| 今月の支出実績   | 当月の支出トランザクション合計（未来日を除く）             |
| 月次予算合計     | 当月に対応する各カテゴリの月次予算（年次予算 ÷ 12）合計    |
| 今月の見込み支出 | 支出実績 + 当月内の未来日トランザクション合計              |
| 予算警告カテゴリ | 進捗率 80% 以上のカテゴリを注意、100% 以上を超過として表示 |
| 最近の取引       | 日付降順で直近 5 件                                        |

**注**: 予算警告カテゴリは展開可能なUIとし、折りたたみ時は警告ラベルのみ、展開時にカテゴリ別の実績/予算を表示する。全カテゴリの詳細は `/budget` に委譲する。

## 10.2 円グラフ

- 年度単位でカテゴリ構成比を表示
  - 収入カテゴリ構成比
  - 支出カテゴリ構成比
- 円グラフは構成比のみを表す

## 10.3 棒グラフ

- カテゴリ別に実績・予算・差分を表示
- 年度単位・月次フィルタの両方に対応

## 10.4 一覧・サマリ

- 年度収支サマリ
- 積立進捗一覧
- カテゴリ別金額・割合・予算差分
- 年度別内部残高推移

## 10.5 イベント別集計

- イベント単位の合計金額
- カテゴリ内訳
- 年度別内訳

## 10.6 年度間比較

- 複数年度の収入・支出・差分の推移をグラフ表示
- カテゴリ別の年度間比較（同一カテゴリの各年度実績）
- 年度ごとの伸び率・減少率の算出

## 10.7 財政体力の推移

- 年度別の内部残高推移をグラフ表示
- イベント単位の影響分析（特定イベントが内部残高に与えた影響を可視化）
- 月次での内部残高推移（オプション）

## 10.8 投資判断支援

- 現在の内部残高表示
- 全積立定義の充足率一覧表示
- 仮想支出額を入力した場合の内部残高シミュレーション
- 積立充足状況を考慮した余裕度の表示
